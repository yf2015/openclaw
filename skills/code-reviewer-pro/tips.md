# Code Reviewer — tips.md
## 代码审查Checklist
### 🐛 正确性
- [ ] 边界条件处理（空值、零、负数、最大值）
- [ ] 错误处理（try-catch、错误码）
- [ ] 并发安全（竞态条件、死锁）
- [ ] 资源释放（文件句柄、数据库连接）

### ⚡ 性能
- [ ] 避免N+1查询
- [ ] 合理使用缓存
- [ ] 避免不必要的循环
- [ ] 大数据量分页处理

### 🔒 安全
- [ ] SQL注入防护（参数化查询）
- [ ] XSS防护（输出转义）
- [ ] 敏感数据不打日志
- [ ] 权限校验

### 📖 可读性
- [ ] 命名表达意图
- [ ] 函数不超过50行
- [ ] 单一职责
- [ ] 必要注释（为什么，不是是什么）
